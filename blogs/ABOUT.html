    # 项目经验

    # 达龙云游戏

    技术栈： VUE2.0/3.0 + WebRTC + WebSocket

    项目描述： 一款通过客户端接收视频流（云端服务器）并实现接收输入设备/触控模拟操作的交互式流媒体（云游戏）平台，主要实现了建联云端服务器、接收视频流、键鼠/手柄输入捕获、多点触控识别等功能模块

    个人职责：

    • 日常维护云游戏项目，处理项目中出现的 bug
    • 对 PC 端和移动端的云游戏项目进行了 VUE2 到 mixin 到 VUE3 的重构，促进了代码可维护性
    • PC 端键鼠/手柄输入映射
    • 移动端多点触控模拟鼠标

    项目难点：

    • 将项目的连接过程使用 websocket 通信进行优化，缩短了连接时间，提升用户体验
    • 利用 requesetAnimationFrame 轮询手柄状态，配合数据映射和 websocket 传输二进制数据

    项目成果： 完成云游戏项目重构与连接优化，网络延迟降低 40%，保障游戏流畅运行；成功接入键鼠手柄输入及多指触控模拟鼠标功能，支持所有游戏操控适配，显著提升玩家操作体验，并在 chinaJoy 游戏大会上获得大量体验者好评

    # 电商营销与运营类落地页

    技术栈： VUE2.0 + Webpack/rollup + Vue-Router + Vuex + VantUI + Sass

    项目描述： 围绕企业高频运营活动需求展开，聚焦于快速开发与维护营销落地页，并将重复性功能基于 VantUI 进行二次封装沉淀出一套轻量级前端组件库

    个人职责：

    • 独立开发并维护各种运营落地页

    • 数据埋点 pv、uv 上报

    • 设计与搭建活动组件库，负责登录、用户账号信息联动、动态表单、选择列表等业务组件的开发

    项目难点：

    • 落地页首屏加载时间过慢的性能定位与优化
    • 通过节流和签名 url、websocket 同步定时器和库存等优化手段配合后端完成秒杀项目
    • 利用虚拟列表和懒加载优化活动参赛名单列表卡顿问题
    • 实现多入口打包、按需加载
    • 通过 slot 自定义表单配置项，提高可扩展性

    项目成果： 基于高频需求搭建了轻量级可复用的前端组件库，有效提升了开发效率与页面维护性；落地页首页加载实现了从 5s 至 1.3s 的优化，提升了用户体验，用户增长量上涨 20%，带动新增用户量从 5.1 万增长至 6.5 万

    # APP 贷款平台 H5 模板包

    技术栈： VUE3.0/React + Vite/Webpack + Pinia + Sass

    项目描述： 这是一款混合开发贷款 APP， 以 H5 为模板包实现跨平台功能，iOS 定制原生外壳，集成用户身份认证、贷款申请及还款流程，兼顾开发效率与系统适配性

    个人职责：

    • 模板包的技术选型和脚手架搭建以及移动端适配方案
    • 基础功能实现，与 iOS 进行跨平台功能交互

    项目难点：

    • 解决移动端的安全区域问题
    • 利用元数据动态调用解决 H5 模板包与 iOS 通信时需要动态调用不同 iOS 包中的方法的问题

    项目成果： 基于 H5 模板包完成混合开发项目，利用 H5 跨平台特性实现代码复用，开发周期缩短 40%；快速搭建功能模块，用户身份认证、贷款还款等核心流程运行稳定，有效降低开发与维护成本

    # 积分商城后台管理系统

    技术栈： VUE2.0 + ElementUI + Axios + Vuex + Vue-Router + Less

    项目描述： 配合积分商城所制作的后台管理系统，包括管理员权限、商品管理、积分管理、订单管理、数据统计等模块

    个人职责：

    • 参与设计系统功能模块，负责权限、商品、积分管理和数据统计模块的实现

    • 项目的 RBAC 角色权限设计与实现

    • Echarts 图表的封装

    项目难点：

    • 结合 canvas 压缩和文件分片上传解决商品图片上传时，图片过大导致上传时间过长的问题
    • 利用 externals 和路由懒加载等优化构建的包，提升项目加载速度

    项目成果： 积分商城后台管理系统通过优化，在运营效率与用户体验上取得一定进展：用户月均活跃时长增长约 25%，商品上架效率提升近 30%，系统响应速度有所改善，用户操作满意度达 80 分以上，为业务运营提供了基础支撑
