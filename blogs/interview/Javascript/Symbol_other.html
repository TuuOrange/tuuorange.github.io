    # Symbol.hasInstance

    可以改变 instanceof 操作默认行为。

    class Stack {
  static [Symbol.hasInstance](obj) {
    return Array.isArray(obj);
  }
}
console.log([] instanceof Stack); // true

    # Symbol.iterator

    指定函数是否会返回对象的迭代器，具有 Symbol.iterator 属性的对象为可迭代对象。

    var num = [1, 2, 3];
for (let i of num) {
  console.log(i);
}
var iterator = num[Symbol.iterator](); // 获取迭代器对象
// iterator.next 将迭代器对象的value属性复制到变量i中
console.log(iterator.next()); // Object{value: 1, done: false}
console.log(iterator.next()); // Object{value: 2, done: false}
console.log(iterator.next()); // Object{value: 3, done: false}
console.log(iterator.next()); // Object{value: undefined, done: true}

    默认情况下，一个自己定义的集合是不可以迭代的，但是我们可以用 Symbol.iterator 使其可迭代

    class List {
  constructor() {
    this.elements = [];
  }

  add(element) {
    this.elements.push(element);
    return this;
  }

  *[Symbol.iterator]() {
    for (let element of this.elements) {
      yield element;
    }
  }
}

let chars = new List();
chars.add("A").add("B").add("C");

// 使用Symbol.iterator实现了迭代
for (let c of chars) {
  console.log(c);
}

// A
// B
// C

    # Symbol.isConcatSpreadable

    当我们使用 concat 对数组进行合并的时候，concat 会将数组拓展为单个元素，但是它会以不同的方式对待单个原始参数：

    let list = {
  0: "JavaScript",
  1: "Symbol",
  length: 2,
};
let message = ["C++"].concat(list);
console.log(message); // ["C++", Object] list对象中的各个元素并没有被合并到数组中

    需要将Symbol.isConcatSpreadable属性添加到 list 对象中：

    let list = {
  0: "JavaScript",
  1: "Symbol",
  length: 2,
  [Symbol.isConcatSpreadable]: true,
};
let message = ["C++"].concat(list);
console.log(message); // ["C++", "JavaScript", "Symbol"]

    # Symbol.toPrimitive

    JS 在每个类型值的原型上定义了 Symbol.toPrimitive 方法，决定对象在进行类型转换时的行为。

    # Symbol.match(regex)

    一个在调用 String.prototype.match() 方法时调用的方法，用于比较字符串。

    # Symbol.replace(regex, replacement)

    一个在调用 String.prototype.replace() 方法时调用的方法，用于替换字符串的子串。

    # Symbol.search(regex)

    一个在调用 String.prototype.search() 方法时调用的方法，用于在字符串中定位子串。

    # Symbol.species(regex)

    用于创建派生对象的构造函数。

    # Symbol.split

    一个在调用 String.prototype.split() 方法时调用的方法，用于分割字符串。

    # Symbol.toStringTag

    一个在调用 String.prototype.toString() 方法时使用的字符串，用于创建对象描述。

    # Symbol.unscopables

    一个定义了一些不可被 with 语句引用的对象属性名称的对象集合。
