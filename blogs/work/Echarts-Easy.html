    # resize 的监听

    利用 window 的 rezise 监听窗口的变化。

    使用节流 Throttle（在 200ms 内调用一次chart.value.resize()），图表的 resize 监听会进行自适应重新绘制图表（图表销毁时要移除监听）

    没有使用防抖的原因：

    1. 防抖不能跟着窗口的变化而实时绘制图表，只会在结束之后重新绘制，用户在拖动窗口的时候并不能看到图表的实时变化
    2. 防抖时间较长（300ms），节流一般是 100-200ms。对于图表重绘这种轻量级的操作，节流能在性能和流畅性之间取得更好的平衡

    # dataset 数据集

    将后端返回的数据通过数据集的dimensions或者encode映射到图表的所需数据中（不需要对数据进行二次遍历包装）

    图表配合axios获取返回的数据（数据结构一致）

    # 优化

    Echarts 自适应大小时，使用 window 监听窗口的大小的变化，但是只能在窗口发生变化时触发；如果使用 v-show 的话，不会触发，图表可能会缩成一团。所以可以使用 ResizeObserver 或者 MutationObserver 来监听盒子 DOM 的尺寸变化。

    可以使用MutationOberserver代替 resize 监听，它可以监听外部盒子元素 DOM 的变化。我们可以通过监听它的尺寸变化来对图表进行重绘。

    使用MutationObserver接口监听DOM树的更改，可以在指定的DOM发生变化时调用
const observer = new MutationObserver((mutationsList) => {
    // 循环寻找我们的echarts dom实例，发现更新的内容包含则调用 resize 方法更新echarts的尺寸
    for (const mutation of mutationsList)
        if (mutation.target === echartDomBox.value) chartObj && chartObj.resize()
})

observer.observe(domBox.value, {
    attributes: true, // 节点的所有属性变化
    childList: false, // 子节点的增删变化
    characterData: true, // 节点的所有字符变化
    subtree: true // 监听整个子树所有节点的变化
})

    observer.disconnect 取消监听，图表销毁时要取消监听哦！

    还可以使用 echarts 的showLoading和hideLoading进行图表更新变化时的加载

    除了 MutationOberser，还有 ResizeObserver 也可以监听 DOM 的尺寸变化

    # 更改图表颜色

    图表的颜色一开始是配置定义好的，并不会监听颜色变化。若要动态改变图表的颜色：

    我们要利用MutationObserver监听 html 的颜色变化，给图表颜色配置var()变量，然后通过监听主题的变化，改变 var 变量，更新图表。

    # 步骤

    首先，书写一个replaceVarStrings中间函数用于将 option 内的所有颜色变量字符串都转换为真正的颜色值，匹配 var()字符串并将中间的颜色提取出来。

    当然我们需要用到宏 api getComputedStyle（会导致重排） 实时动态计算获取我们的 dom 实例的样式属性，其返回值的 getPropertyValue 方法传入变量名称字符串就会返回实时的变量颜色值

    /**
 * echarts样式
 */
export const useThemeValue = (styleVariables: string) => {
  return getComputedStyle(document.documentElement).getPropertyValue(styleVariables);
};

export function replaceVarStrings(obj: Record<string, any>) {
  const newObj: Record<string, any> = Array.isArray(obj) ? [] : {};
  for (const key in obj) {
    if (typeof obj[key] === "object") {
      newObj[key] = replaceVarStrings(obj[key]); // 递归处理子对象
    } else if (typeof obj[key] === "string" && obj[key].startsWith("var(") && obj[key].endsWith(")")) {
      const varContent = obj[key].slice(4, -1); // 提取括号内的内容
      newObj[key] = useThemeValue(varContent); // 替换为括号内的内容
    } else {
      newObj[key] = obj[key]; // 其他情况直接复制值
    }
  }

  return newObj;
}
